Introduction

This is a visualization of gene expression patterns of the Siphonophore Bargmannia elongata, showing normalized counts between two major tissue types. I generated normalized counts using the R package edgeR, and generated Gene Ontology (GO) terms using the program blast2go . Under this method, genes are grouped into three main domains: 1) cellular component; 2) molecular function; 3) biological proccess.
